## Environment

- Create virtual environment for this project
```shell=
python3 -m venv venv
```
- Activate virtual environment
```shell=
source venv/bin/activate
```
- Install

```shell=
pip install -r requirements.txt
```

## Execute

```shell=
cd bank
flask run
```

## Note

`flask init-db`
`export FLASK_APP=app.py`
